# AUY1105 - Infraestructura como Código II

## Descripción
Este repositorio contiene la implementación de un pipeline automatizado para análisis de calidad y seguridad en código de infraestructura utilizando Terraform y GitHub Actions.

## Objetivos
- Implementar infraestructura en AWS con Terraform
- Automatizar validaciones de calidad y seguridad
- Aplicar políticas de seguridad mediante OPA
- Utilizar buenas prácticas de control de versiones con GitHub

## Tecnologías utilizadas
- Terraform
- AWS
- GitHub Actions
- TFLint
- Checkov
- Open Policy Agent (OPA)

## Estructura del proyecto
- Código Terraform
- Workflows de automatización
- Políticas de seguridad
